Q&A

How long does it take?

I'm not paying for a designated server so it can take 10-15 seconds for the server to wake up. It then takes Scrabble Oracle around 20-30 seconds to find the highest scoring word. On average the whole process takes well under a minute.

Can my rack/board have blanks?

Not right now but if enough people request it I might add that feature.
